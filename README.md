# GrowthPath - Competence Management System

GrowthPath is a streamlined competence management platform that helps employees and managers identify, track, and develop key skills. It provides a clear framework for skill assessment and creates a foundation for targeted professional growth and strategic talent management.

## Overview

GrowthPath empowers organizations to:
- **Identify** critical competencies across roles and departments
- **Track** skill development and proficiency levels
- **Develop** targeted learning paths and career progression plans
- **Align** individual growth with organizational goals

## Core Features

### 1. Competency Framework
- Comprehensive skill taxonomy with customizable categories
- Role-specific competency models
- Multi-level proficiency scales (Beginner → Intermediate → Advanced → Expert)
- Support for both technical and soft skills

### 2. Assessment & Evaluation
- Employee self-assessments
- Manager evaluations and feedback
- 360-degree feedback capability
- Skill gap analysis and visualization
- Historical tracking and progress monitoring

### 3. Development Planning
- Personalized learning paths
- SMART goal setting and tracking
- Training and course recommendations
- Mentorship program integration
- Career progression roadmaps

### 4. Analytics & Insights
- Individual skill profiles and dashboards
- Team competency heatmaps
- Organization-wide skill gap analysis
- Succession planning support
- Training ROI metrics

### 5. User Roles
- **Employees**: Self-assessment, view development plans, track progress
- **Managers**: Assess team members, approve goals, provide feedback
- **HR/Admins**: Configure system, manage competencies, access analytics
- **Leadership**: Strategic oversight, talent analytics, workforce planning

## Technology Stack

- **Frontend**: React 18 + Vite
- **Backend**: FastAPI (Python)
- **Database**: SQLite (development) / PostgreSQL (production)
- **ORM**: SQLAlchemy

## Getting Started

### Quick Start

**On macOS/Linux:**
```bash
./start-all.sh
```

**On Windows:**
```bash
start-all.bat
```

This will start both backend and frontend automatically!

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs

See [SETUP.md](SETUP.md) for detailed setup instructions and manual installation.
